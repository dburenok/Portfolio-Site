Scanbot is an Arduino-ESP32 based 360-degree point-cloud scanner, programmed in C++. It pairs to a computer over bluetooth and performs a full 3D scan of the environment. Watch it in action!

  • 3D point-cloud scanner built and programmed in C++ using Arduino and Garmin LIDAR-Lite
  • Java-based companion app for point-cloud visualization
  • Professional enclosure design and easy-to-navigate LCD menu system

The code for Scanbot is available on my GitHub.

 Teletext is an internal communication system developed for Atlantic Regional Council.

The goal was to replace an existing system in order to save on cost and achieve a simplfied user interface.

Teletext is powered by Node.js, Express.js, Passport.js, and uses MongoDB for storage. The app was tested with hundreds of users, and remains in continuous development.

Spellchecker App

 BeautySpell is a GUI Java application that performs spell-checking using an custom path-finding algorithm.

BeautySpell makes extensive use of Object Oriented Design principles to build a flexible, persistent user program which handles, saves, and retrives custom-defined classes and objects.

  • Spell-checking application utilizing custom algorithm and following OOP design principles (Java)
  • Rigorous testing with JUnit 5
  • Clear and systematic code documentation

The code for BeautySpell is available on my GitHub.
